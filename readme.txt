The Narnoo operator plugin allows a tourism business to manage and display their promotional product media hosted on Narnoo.com from their WordPress website.

== Description ==

The Narnoo operator plugin allows a tourism business to manage and display their promotional, product media hosted on Narnoo.com from their WordPress website. Using the plugin Operators can manage their media on Narnoo.com and also display their media through their own website's blogs or pages. Narnoo provides shortcodes that make displaying image galleries, videos and brochures simple and easy to achieve. All media files are hosted on Narnoo's global servers therefore, freeing up the users server requirements.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Retrieve your APP authentication keys from your account at Narnoo.com. You will need your Access Key and corresponding Secret Key. Log in to your Narnoo.com account -> My Account -> either create a New APP or View Applications.
3. Enter your APP keys in Settings -> Narnoo API -> API  Settings area.
4. Your WordPress site is now connected to your Narnoo.com account.  

== Frequently asked questions ==

How to I get my Narnoo APP keys?

You can retrieve your APP keys from your account on Narnoo.com. Simply log into your account and go to the My Account -> View Applications menu option. From here you can either create or view existing key pairs.
